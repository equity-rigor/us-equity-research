# Retail translation glossary

Use to convert institutional language to plain Chinese for retail / 非专业版 deliverables.

## Headline / 仓位语言

| 机构语言 | 零售语言 |
|---|---|
| 减仓 -25% | 卖一部分 (大约四分之一) |
| 减仓 -50% | 卖一半 |
| 持有 / 不动 | 你已经买的话先不动 |
| 加仓 +25% | 可以再买一些 (大约现有头寸的四分之一) |
| 建仓 5-10% | 可以小买一点 (占你股票总投资的 5-10%) |
| 中位预期收益 +0.5% | 平均来看股价大概率持平 (略涨) |
| 中位预期收益 -12.4% | 平均来看股价大概率下跌 12% 左右 |
| 情景加权区间 [-15%, -3%] | 多数情况股价会跌 3%-15% |
| 信号倾向 中度偏空 | 我比较看空 (但不是极度看空) |
| 信号倾向 中性 | 我没什么特别看法, 看不清方向 |
| Source-conditional 中性轻偏多 | 我倾向看好但前提是几个关键消息要兑现 — 现在还不能下结论 |

## 估值语言

| 机构语言 | 零售语言 |
|---|---|
| EPS | 公司每股能赚多少 |
| PE / 市盈率 | 市场愿意为公司每赚一块钱付多少 |
| EPS × PE | 用每股利润 × 市场倍数 = 大概值多少钱 |
| SOTP / 分部加总 | 把公司各业务分别估值再相加 |
| P/B / 市净率 | 股价相对于公司"账面值"高多少倍 |
| EV/EBITDA | 公司整体估值 / 公司未含利息税款的盈利 |
| FCF yield | 公司能产生多少自由现金 / 市值 |
| Trough multiple / 历史低估 | 历史上最低的时候是多少倍 |
| Forward EPS / 前瞻 EPS | 我对未来某年公司能赚多少的估计 |
| Bear EPS / 空头 EPS | 如果情况比较差公司能赚多少 |
| Bull EPS / 多头 EPS | 如果情况比较好公司能赚多少 |
| Base EPS / 基础 EPS | 最可能的情况公司能赚多少 |

## 锚点 / 来源语言

| 机构语言 | 零售语言 |
|---|---|
| 锚点 (anchor) | 关键事件 / 关键信号 |
| Top-3 anchors | 最重要的三个判断依据 |
| Source-conditional | 依赖关键信息核实的 |
| S1 (年报) | 公司年报里写的 (最可信) |
| S2 (公告 / 季报) | 公司公开公告里写的 (可信) |
| S3 (IR / 投资者沟通) | 公司在投资者会议上说的 (可信度中等) |
| S4 (卖方一致) | 卖方分析师的一致看法 (有偏差) |
| S5 (行业数据) | 行业研究机构的数据 (有偏差) |
| Pending | 还没核实的消息 (不能依赖) |
| Anchor verification | 等关键消息得到确认 |

## 概率框架语言

| 机构语言 | 零售语言 |
|---|---|
| 5 情景概率分布 | 5 种可能情况, 每种发生的可能性不一样 |
| 强多 (强 bull) | 最乐观的情况 (可能性 [X]%) |
| 多头 | 比较乐观的情况 |
| 基础情景 | 最可能发生的情况 |
| 空头 | 比较悲观的情况 |
| 强空 | 最悲观的情况 |
| Probability shift -10pp / +10pp | 可能性下调 10%/上调 10% |
| Conviction tag | 我的看法把握度 |

## 仓位 / 风险语言

| 机构语言 | 零售语言 |
|---|---|
| σ / volatility / 波动率 | 这只股票一般起落多大 |
| Annual σ 20% | 一年内涨跌 20% 是正常的 |
| Sharpe / 夏普比率 | 风险调整后回报, 越高越值得做 |
| Kelly / Kelly 公式 | 用数学算出该投多少最划算 |
| Conviction multiplier 0.25× | 因为不太确定, 实际投入打 1/4 折 |
| A0 tail risk / 尾部风险 | 极端坏事件 (发生概率小但影响大) |
| Bull tail / Bear tail | 极端好事件 / 极端坏事件 |

## 触发器 / 行动语言

| 机构语言 | 零售语言 |
|---|---|
| What would reverse | 什么会让我改变看法 |
| Numerical denominator / 数值阈值 | 具体的数字标准 |
| Catalyst calendar | 关键时间表 |
| Re-sizing trigger | 重新评估仓位的时点 |
| Action mapping | 触发后该怎么做 |

## 整段示例

**机构原句**:
> "本意见 source-conditional 中性轻偏多 / 默认持有, 12M 中位预期 +0.5%, 情景加权区间 [-2%, +3%]; 由于核心三锚点 (L1/L2/B2) 均为 S3-S5 等级, conviction multiplier 取 0.15×, Kelly 调整后 ≈ 0.2% (trivial). 再评估触发: 1Q26 季报披露 OLED IT 出货占比 ≥10% 或半年报 LCD 价格弹性印证 (L1)."

**零售翻译**:
> "我对这只股票的看法: 倾向略微看好, 但前提是几个关键消息(LCD 价格、OLED IT 出货量、新厂投产时点)未来几个季度要被确认. 现在不能下定论, 所以**如果你已经持有, 先不要动; 如果还没买, 先不急着买**. 平均来看一年股价大概率持平, 多数情况会在 -2% 到 +3% 之间. 等到 4 月 1Q26 季报或者 9 月半年报出来再看 — 如果 OLED IT 业务出货占比超过 10%, 或者 LCD 价格走势如预期, 我就会改成更看好."

## Cautious framing — 永远附加

零售对话 script 必须在结尾包含:

> "这是基于现有信息的分析, 不构成投资建议. 任何股票都有亏损可能. 请根据自己的财务状况和风险承受能力决定."

不要承诺收益. 不要保证下跌不会发生. 强调"概率"而不是"必然".

## 常见 retail Q&A — 这些问题要预留答案

不能回避的 15 个问题 (具体回答模板见 `<ticker>_零售_对话Script_<author>.docx`):

1. 现在能买吗?
2. 一定会跌/涨吗?
3. 国家会扶持吗?
4. 卖方都说买入, 你和他们谁对?
5. 我已经买了怎么办?
6. 我不卖会怎样?
7. 中位 -X% 是什么意思?
8. 一年最坏会怎样?
9. 一年最好会怎样?
10. 明天能涨吗? (耐心解释短期不可预测)
11. 有内部消息吗? (no — 强调一切来源公开)
12. 我应该多关注什么?
13. 你是不是太悲观/乐观了?
14. 什么情况下你会改看法?
15. [名分析师/基金经理] 都买了, 我能跟着买吗? (解释跟单风险)
